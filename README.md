# immersive-event-planning-deck
A card deck system for planning immersive arts events. Designed as part Film Hub Midlands' Broader Screen resources.

## What it is

A digital and physical planning toolkit, designed for cinema exhibition venue programmers, curators, producers, technical staff and other venue workers to discuss and plan immersive events presenting Broader Screen content (VR, XR, interactive media, games, etc.).

Produced by Broadway's [Near Now](http://nearnow.org.uk) for [Film Hub Midlands](https://filmhubmidlands.org) as part of the BFI Film Audience Network's Broader Screen resources.

## Features

- **Planning cards** across 6 categories: Vision & concept, Space & atmosphere, Audience, Operations, Marketing & comms, Risk & quality
- **Flip cards** to reveal key questions and discussion prompts
- **3 facilitation modes**: Browse, Journey map, Solidarity round
- **Add / edit / delete cards** — all via the UI, no code needed
- **Print-ready PDF** — A5 or A6, front & back, straight from the browser
- **Edits persist** — changes are saved in the browser (localStorage) and carry over to print
- **Filter by category** — focus discussion on a single planning area
- **Shuffle** — randomise for spontaneous draw exercises
